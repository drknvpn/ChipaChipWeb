# ДЕМО РЕДИЗАЙНА ДЛЯ ОФИЦИАЛЬНОГО САЙТА ИСПОЛНИТЕЛЯ ChipaChip.

![Стаус проекта](https://img.shields.io/badge/status-demo%20portfolio-blue)
![Лицензия](https://img.shields.io/badge/license-non--commercial%20demo-green)

## 📋 Описание

Данный проект представляет собой демонстрационную версию НЕофициального сайта исполнителя **ChipaChip**. Сайт разработан в рамках портфолио веб-разработчика с целью демонстрации навыков frontend-разработки и современного подхода к веб-дизайну.

**ДЕМО:** [ПРОЕКТ НА VERCEL](https://chipa-chip-web.vercel.app)

---

## ⚠️ Важная информация

### Правовой статус проекта

Данный проект:
- ✅ **Не является коммерческим продуктом**
- ✅ **Не претендует на звание официального сайта**
- ✅ **Создан исключительно в образовательных целях**

## 🎯 Цели проекта

1. **Развитие технических навыков** — применение современных технологий веб-разработки
2. **Демонстрация UX/UI решений** — создание удобного и эстетичного интерфейса
3. **Портфолио разработчика** — демонстрация профессиональных компетенций
4. **Фан-проект** — выражение уважения творчеству артиста

---

## 🛠 Технологии

### Frontend
- **React 18** — библиотека для создания интерфейсов
- **TypeScript** — типизация и надежность кода
- **Vite** — современный сборщик проектов
- **CSS3** — стилизация и анимации

### Особенности реализации
- **Адаптивный дизайн** — корректное отображение на всех устройствах
- **PWA (Progressive Web App)** — возможность установки как приложение
- **Оптимизация производительности** — быстрая загрузка и плавная работа
- **SEO-оптимизация** — правильная семантика и мета-теги
- **Доступность (a11y)** — соблюдение стандартов доступности

---

## 🎨 Ключевые функции

### Интерфейс
- Современный минималистичный дизайн в темной теме
- Плавные анимации и переходы
- Фиксированная навигация с эффектом blur
- Адаптивное мобильное меню

### Контент
- Раздел с биографией исполнителя
- Актуальное расписание концертов
- Интеграция с билетными площадками (Яндекс Афиша, Афиша, Kassir, ticketscloud)
- Разделы для музыки и видео
- Ссылки на социальные сети

### Технические решения
- Параллакс-эффект для главного изображения
- Ленивая загрузка ресурсов
- Оптимизация изображений
- Корректная работа с шрифтами (Google Fonts)
